data(maits, package='MAST')

sca_raw <- FromMatrix(maits$expressionmat %>% t() %>% unlog() , maits$cdat, maits$fdat)
sca_filt <- sca_raw[freq(sca_raw)>.1,] %>% subset(., ourfilter)
sca_filt <- sca_filt[sample(nrow(sca_filt), 1500),]

Potential benefits:

  • Improve and speed analyses â€“ mix and match normalization methods, dimensionality reduction, plotting
  • Improve statistical methods â€“ allow inter-method comparison
  • Improve software â€“ write utility functions once, and well

Potential pitfalls:

  • What is in slot 1 of assays? Counts? Normalized counts? Log-transformed?
  • Implicit contracts between functions (load â€“ normalize â€“ analysis )
  • Idiosyncratic results ( analysis â€“ plot )
